# Welcome to Frosted UI

**Quick links:** [GitHub](https://github.com/aussieljk/frosted) · [llms.txt](/llms.txt)

Frosted is a design system heavily based on [Radix Themes](https://www.radix-ui.com/).

The library is still a work in progress so you might expect some breaking changes.

If you have any questions or suggestions, feel free to open an issue on [GitHub](https://github.com/aussieljk/frosted).
