# Tooltip

A short label shown on hover or focus.

<Demo id="tooltip" />

## Usage

Tooltip wraps its trigger element and takes the label as the `content` prop:

```tsx
import { Button, Tooltip } from '@aussieljk/frosted';

<Tooltip content="Add to library">
  <Button variant="surface">Hover me</Button>
</Tooltip>;
```
