# Dialog

A modal window overlaid on the page.

<Demo id="dialog" />

## Usage

```tsx
import { Button, Dialog } from '@aussieljk/frosted';

<Dialog.Root>
  <Dialog.Trigger nativeButton>
    <Button>Open</Button>
  </Dialog.Trigger>
  <Dialog.Content style={{ maxWidth: 450 }}>
    <Dialog.Title>Title</Dialog.Title>
    <Dialog.Description>Description text.</Dialog.Description>
    <Dialog.Close>
      <Button variant="soft" color="gray">
        Close
      </Button>
    </Dialog.Close>
  </Dialog.Content>
</Dialog.Root>;
```

## API reference

### Dialog.Root

State container for the dialog.

<PropsTable component="Dialog.Root" />

### Dialog.Trigger

Opens the dialog; wraps your own element (set `nativeButton` when the child is a button).

<PropsTable component="Dialog.Trigger" />

### Dialog.Content

The modal panel.

<PropsTable component="Dialog.Content" />

### Dialog.Title

Accessible title, rendered as a `Heading`.

<PropsTable component="Dialog.Title" />

### Dialog.Description

Accessible description, rendered as a `Text`.

<PropsTable component="Dialog.Description" />

### Dialog.Close

<PropsTable component="Dialog.Close" />
