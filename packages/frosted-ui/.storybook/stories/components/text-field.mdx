# Text Field

A single-line text input, with optional leading/trailing slots.

<Demo id="text-field" />

## Usage

```tsx
import { TextField } from '@aussieljk/frosted';

<TextField.Root>
  <TextField.Slot>
    <SearchIcon />
  </TextField.Slot>
  <TextField.Input placeholder="Search the docs…" />
</TextField.Root>;
```

## API reference

### TextField.Root

Container that provides the field's visual style.

<PropsTable component="TextField.Root" />

### TextField.Input

The native input. Also accepts all standard `input` attributes (`placeholder`, `type`, …).

<PropsTable component="TextField.Input" />

### TextField.Slot

Decorations (icons, buttons) before or after the input.

<PropsTable component="TextField.Slot" />
