# Avatar

A profile image with an automatic fallback.

<Demo id="avatar" />

## Usage

```tsx
import { Avatar } from '@aussieljk/frosted';

<Avatar size="4" src="https://example.com/me.jpg" fallback="LK" />;
```

The `fallback` (required) renders while the image loads or when `src` is missing/broken.

## Props

Avatar also accepts the native `img` attributes (`src`, `alt`, …).

<PropsTable component="Avatar" />

Related: `AvatarGroup` and `AvatarStack` for overlapping collections of avatars.
