# Icons

Frosted UI ships no icon set of its own — bring the icon library you already use.
Components render a small set of built-in inline icons (checkmarks, chevrons, …)
out of the box, and an icon adapter can swap those for your library so everything
matches.

### 1. Install an icon library

Pick one of the supported libraries and install it:

- `lucide-react`
- `@phosphor-icons/react`
- `@heroicons/react`
- `@tabler/icons-react`
- `@hugeicons/react` + `@hugeicons/core-free-icons`

### 2. Register the adapter (optional)

Import the matching adapter once (e.g. in your app entry) so the built-in
component icons render from your library:

```tsx
import '@aussieljk/frosted/icons/lucide';
```

Or scope it to a subtree with `IconProvider`:

```tsx
import { IconProvider } from '@aussieljk/frosted';
import { lucideAdapter } from '@aussieljk/frosted/icons/lucide';

<IconProvider library={lucideAdapter}>
  <App />
</IconProvider>;
```

### 3. Use icons directly

For your own UI, import icons straight from the library:

```tsx
import { Store } from 'lucide-react';

function MyComponent() {
  return <Store size={16} />;
}
```
