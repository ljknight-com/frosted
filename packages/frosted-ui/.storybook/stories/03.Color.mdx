# Color system

The color palettes in Frosted UI are the [Tailwind CSS v4 palettes](https://tailwindcss.com/docs/colors),
each extended with an extra light shade to form a 12-step scale with light, dark and alpha variants.

Colors are grouped into 3 categories: `accents`, `grays`, and `semantic` (info, success, warning, danger).
All of these can be specified on your `<Theme />`, or per component where appropriate.

## Understanding the scale

Each scale has 12 steps with dedicated roles — 1-2 backgrounds, 3-5 component backgrounds,
6-8 borders, 9-10 solid colors, 11-12 text. In light mode the steps map to the Tailwind stops
(an extra light shade, then 50 through 950); dark mode reads the same stops in reverse with the
solid steps anchored near 500-600 so buttons stay recognizable.
