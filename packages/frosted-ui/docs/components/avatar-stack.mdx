# Avatar Stack

Stack overlapping avatars with the first child on top.

<Demo id="avatar-stack" />

## Usage

```tsx
import { AvatarStack } from '@aussieljk/frosted';

<AvatarStack.Root size="3">
  <AvatarStack.Avatar src="/a.png" fallback="Ada Lovelace" />
  <AvatarStack.Avatar src="/b.png" fallback="Grace Hopper" />
</AvatarStack.Root>;
```

Children are rendered in reverse DOM order so that stacking follows source order visually.

## API reference

### AvatarStack.Root

Sets the `size` for every avatar inside.

<PropsTable component="AvatarStack.Root" />

### AvatarStack.Avatar

An avatar within the stack; its `size` is inherited from the stack.

<PropsTable component="AvatarStack.Avatar" />

## Related

`Avatar` for a single avatar and `AvatarGroup` for an evenly overlapping row.
