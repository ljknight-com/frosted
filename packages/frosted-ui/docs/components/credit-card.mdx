# CreditCard

Display a flippable payment card with front artwork and back details.

<Demo id="credit-card" />

## Usage

```tsx
import { CreditCard } from '@aussieljk/frosted';

<CreditCard.Root defaultFace="front">
  <CreditCard.Content>
    <CreditCard.Front>
      <CreditCard.FrontHeader>
        <CreditCard.Logo>frosted</CreditCard.Logo>
        <CreditCard.Brand>
          <CreditCard.BrandLogo brand="visa" />
        </CreditCard.Brand>
      </CreditCard.FrontHeader>
      <CreditCard.FrontFooter>
        <CreditCard.Title>Claude credits</CreditCard.Title>
        <CreditCard.LastFour>•••• 0991</CreditCard.LastFour>
      </CreditCard.FrontFooter>
    </CreditCard.Front>
    <CreditCard.Back>
      <CreditCard.MagStripe />
      <CreditCard.BackContent>…</CreditCard.BackContent>
    </CreditCard.Back>
  </CreditCard.Content>
  <CreditCard.Trigger>Flip card</CreditCard.Trigger>
</CreditCard.Root>;
```

## API reference

The core parts are below. The front is composed with `FrontHeader` / `FrontFooter` / `Logo` / `Brand` / `Title` / `LastFour`, and the back with `MagStripe` / `BackContent` / `Fieldset` / `FieldGroup` / `Field` / `FieldLabel` / `FieldError` / `TextField` — see the stories for full compositions, including editable payment forms.
