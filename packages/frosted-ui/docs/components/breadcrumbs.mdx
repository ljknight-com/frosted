# Breadcrumbs

Show the current page's location within a navigation hierarchy.

<Demo id="breadcrumbs" />

## Usage

```tsx
import { Breadcrumbs } from '@aussieljk/frosted';

<Breadcrumbs.Root>
  <Breadcrumbs.Item render={<a href="/" />} nativeButton={false}>
    Home
  </Breadcrumbs.Item>
  <Breadcrumbs.Item render={<a href="/products" />} nativeButton={false}>
    Products
  </Breadcrumbs.Item>
  <Breadcrumbs.Item>Sneaker Bot</Breadcrumbs.Item>
</Breadcrumbs.Root>;
```

Collapse long trails with `Breadcrumbs.Dropdown`, which renders an ellipsis trigger
containing `Breadcrumbs.DropdownItem` entries.

## API reference

### Breadcrumbs.Root

Container for the breadcrumb trail.
