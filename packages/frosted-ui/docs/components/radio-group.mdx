# Radio Group

Select a single option from a list.

<Demo id="radio-group" />

## Usage

```tsx
import { RadioGroup } from '@aussieljk/frosted';

<RadioGroup.Root defaultValue="1">
  <RadioGroup.Item value="1">Default</RadioGroup.Item>
  <RadioGroup.Item value="2">Comfortable</RadioGroup.Item>
  <RadioGroup.Item value="3">Compact</RadioGroup.Item>
</RadioGroup.Root>;
```

## API reference

### RadioGroup.Root

State container for the group — holds the selected value, size, and color.

<PropsTable component="RadioGroup.Root" />

### RadioGroup.Item

A single radio button with an optional label.

<PropsTable component="RadioGroup.Item" />

## Related
