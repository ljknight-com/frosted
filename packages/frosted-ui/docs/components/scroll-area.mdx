# ScrollArea

Scroll overflowing content with themed, cross-browser scrollbars.

<Demo id="scroll-area" />

## Usage

```tsx
import { ScrollArea } from '@aussieljk/frosted';

<ScrollArea type="hover" scrollbars="vertical" style={{ height: 160 }}>
  <LongContent />
</ScrollArea>;
```

The `type` prop controls scrollbar visibility (similar to macOS scrollbar preferences), and `scrollbars` limits which axes scroll.
