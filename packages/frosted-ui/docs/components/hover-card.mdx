# Hover Card

A rich preview card shown when hovering a link or other element.

<Demo id="hover-card" />

## Usage

```tsx
import { HoverCard, Link, Text } from '@aussieljk/frosted';

<HoverCard.Root>
  <HoverCard.Trigger>
    <Link href="https://example.com">Hover me</Link>
  </HoverCard.Trigger>
  <HoverCard.Content>
    <Text size="2">Preview content.</Text>
  </HoverCard.Content>
</HoverCard.Root>;
```

## API reference

### HoverCard.Root
