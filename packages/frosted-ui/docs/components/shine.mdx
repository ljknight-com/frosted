# Shine

Apply an interactive specular highlight that follows the pointer across its children.

<Demo id="shine" />

## Usage

```tsx
import { Shine } from '@aussieljk/frosted';

<Shine puffyness="1.5">
  <Badge color="yellow">Premium</Badge>
</Shine>;
```

Move the pointer over the page to see the highlight travel across the content. Higher `puffyness` spreads the highlight over softer, "puffier" edges.
