# Fieldset

Group related fields under a shared legend.

<Demo id="fieldset" />

## Usage

```tsx
import { Field, Fieldset, TextField } from '@aussieljk/frosted';

<Fieldset.Root>
  <Fieldset.Legend>Billing Details</Fieldset.Legend>
  <Field.Root name="company">
    <Field.Label>Company</Field.Label>
    <TextField.Root>
      <TextField.Input placeholder="Acme Inc." />
    </TextField.Root>
  </Field.Root>
</Fieldset.Root>;
```

## API reference

### Fieldset.Root

Renders a native `fieldset` element. Set `disabled` to disable every field inside at once.

<PropsTable component="Fieldset.Root" />

### Fieldset.Legend

A caption for the group. Use `variant="label"` for a smaller, field-label-sized legend.

<PropsTable component="Fieldset.Legend" />

## Related

`Field` for individual controls and `Form` for whole-form submission.
