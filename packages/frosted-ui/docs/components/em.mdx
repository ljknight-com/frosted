# Em

Emphasize a word or phrase with italics.

<Demo id="em" />

## Usage

```tsx
import { Em, Text } from '@aussieljk/frosted';

<Text>
  We <Em>had</Em> to do something about it.
</Text>;
```

## Props

Em renders the native `em` element and has no props of its own — standard `em` attributes are supported.

## Related

`Strong` for strong importance rather than stress emphasis.
