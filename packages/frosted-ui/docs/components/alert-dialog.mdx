# Alert Dialog

A modal confirmation dialog that interrupts the user with an important choice.

<Demo id="alert-dialog" />

## Usage

```tsx
import { AlertDialog, Button } from '@aussieljk/frosted';

<AlertDialog.Root>
  <AlertDialog.Trigger>
    <Button color="red">Revoke access</Button>
  </AlertDialog.Trigger>
  <AlertDialog.Content style={{ maxWidth: 450 }}>
    <AlertDialog.Title>Revoke access</AlertDialog.Title>
    <AlertDialog.Description>Are you sure?</AlertDialog.Description>
    <AlertDialog.Cancel>
      <Button variant="soft" color="gray">
        Cancel
      </Button>
    </AlertDialog.Cancel>
    <AlertDialog.Action>
      <Button color="red">Revoke access</Button>
    </AlertDialog.Action>
  </AlertDialog.Content>
</AlertDialog.Root>;
```

## API reference

### AlertDialog.Root

State container for the dialog.

<PropsTable component="AlertDialog.Root" />

### AlertDialog.Trigger

Opens the dialog; wraps your own element.

<PropsTable component="AlertDialog.Trigger" />

### AlertDialog.Content

The modal panel.

<PropsTable component="AlertDialog.Content" />

### AlertDialog.Title

Accessible title, rendered as a `Heading`.

<PropsTable component="AlertDialog.Title" />

### AlertDialog.Description

Accessible description, rendered as a `Text`.

<PropsTable component="AlertDialog.Description" />

### AlertDialog.Action

Closes the dialog while confirming the action; wraps your own element.

<PropsTable component="AlertDialog.Action" />

### AlertDialog.Cancel

Closes the dialog without confirming; wraps your own element.

<PropsTable component="AlertDialog.Cancel" />

### AlertDialog.Close

Generic close; wraps your own element (alternative to `Action`/`Cancel`).

<PropsTable component="AlertDialog.Close" />
