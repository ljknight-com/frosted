# VisuallyHidden

Hide content visually while keeping it available to screen readers.

<Demo id="visually-hidden" />

## Usage

```tsx
import { VisuallyHidden } from '@aussieljk/frosted';

<IconButton>
  <MagnifyingGlassIcon />
  <VisuallyHidden>Search</VisuallyHidden>
</IconButton>;
```

## Props

VisuallyHidden renders a `span` by default (override via the `render` prop); custom
`style` is merged on top of the hiding styles. The raw `VISUALLY_HIDDEN_STYLES`
object is also exported for spreading onto any element directly.
