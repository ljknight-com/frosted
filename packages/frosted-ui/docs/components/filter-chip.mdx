# FilterChip

Toggle a filter on or off.

<Demo id="filter-chip" />

## Usage

FilterChip is a styled checkbox — it supports `checked` / `defaultChecked`, submits its value with forms via
`name`, and renders its children (text and/or an icon) inside the chip:

```tsx
import { FilterChip } from '@aussieljk/frosted';

<FilterChip name="availability" defaultChecked>
  In stock
</FilterChip>;
```
