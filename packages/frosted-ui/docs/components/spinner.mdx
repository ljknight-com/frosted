# Spinner

Indicate that content or an action is loading.

<Demo id="spinner" />

## Usage

```tsx
import { Spinner } from '@aussieljk/frosted';

<Spinner size="3" />

// Swap children for a spinner while `loading` is true —
// their dimensions are preserved and interaction is disabled.
<Spinner loading={isLoading}>
  <Switch defaultChecked />
</Spinner>;
```
