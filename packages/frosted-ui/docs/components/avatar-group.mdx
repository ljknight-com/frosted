# Avatar Group

Show a row of overlapping avatars, e.g. the members of a group.

<Demo id="avatar-group" />

## Usage

```tsx
import { AvatarGroup } from '@aussieljk/frosted';

<AvatarGroup.Root size="3">
  <AvatarGroup.Avatar src="/a.png" fallback="Ada Lovelace" />
  <AvatarGroup.Avatar src="/b.png" fallback="Grace Hopper" />
</AvatarGroup.Root>;
```

## API reference

### AvatarGroup.Root

Sets the `size`, `shape`, and `color` for every avatar inside.

<PropsTable component="AvatarGroup.Root" />

### AvatarGroup.Avatar

An avatar within the group; its `size` and `shape` come from the root, not per-avatar props.

<PropsTable component="AvatarGroup.Avatar" />

## Related

`Avatar` for a single avatar and `AvatarStack` for a stack where the first child appears on top.
