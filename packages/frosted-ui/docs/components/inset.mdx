# Inset

Bleed content into the padding of a parent container.

<Demo id="inset" />

## Usage

```tsx
import { Card, Inset } from '@aussieljk/frosted';

<Card>
  <Inset side="top" pb="current">
    <img src="/cover.png" alt="" />
  </Inset>
  <Text>Card text with normal padding.</Text>
</Card>;
```

## Props

Inset extends the native `div` element — standard div attributes are also supported.
Padding props accept `'current'` (keep the parent's padding) or `'0'` (remove it).
