# Blockquote

Display a block-level quotation.

<Demo id="blockquote" />

## Usage

```tsx
import { Blockquote } from '@aussieljk/frosted';

<Blockquote>
  Perfection is achieved, not when there is nothing more to add, but when there is nothing left to take away.
</Blockquote>;
```
