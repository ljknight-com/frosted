# WidgetStack

Flip through a deck of widgets, one card at a time.

<Demo id="widget-stack" />

## Usage

```tsx
import { WidgetStack } from '@aussieljk/frosted';

<WidgetStack.Root orientation="horizontal">
  <WidgetStack.Prev>{'<'}</WidgetStack.Prev>
  <WidgetStack.Stack style={{ width: 320, height: 160 }}>
    <WidgetStack.Item>…</WidgetStack.Item>
    <WidgetStack.Item>…</WidgetStack.Item>
  </WidgetStack.Stack>
  <WidgetStack.Next>{'>'}</WidgetStack.Next>
</WidgetStack.Root>;
```

## API reference

### WidgetStack.Root

State container for the stack.

<PropsTable component="WidgetStack.Root" />

### WidgetStack.Stack

The sized viewport holding the items; swipe or scroll to move between them.

<PropsTable component="WidgetStack.Stack" />

### WidgetStack.Item

One widget in the stack. Its children fill the item — size them with `width: 100%` and `height: 100%`.

### WidgetStack.Prev

Moves to the previous item.

<PropsTable component="WidgetStack.Prev" />

### WidgetStack.Next

Moves to the next item.

<PropsTable component="WidgetStack.Next" />
