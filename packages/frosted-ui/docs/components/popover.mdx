# Popover

A floating panel anchored to a trigger, for rich interactive content.

<Demo id="popover" />

## Usage

```tsx
import { Button, Popover } from '@aussieljk/frosted';

<Popover.Root>
  <Popover.Trigger>
    <Button variant="soft">Comment</Button>
  </Popover.Trigger>
  <Popover.Content style={{ width: 360 }}>
    …
    <Popover.Close>
      <Button size="1">Done</Button>
    </Popover.Close>
  </Popover.Content>
</Popover.Root>;
```
