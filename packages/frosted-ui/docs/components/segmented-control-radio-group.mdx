# SegmentedControlRadioGroup

Pick a single option from a segmented group, backed by a form-friendly radio input.

<Demo id="segmented-control-radio-group" />

## Usage

```tsx
import { SegmentedControlRadioGroup } from '@aussieljk/frosted';

<SegmentedControlRadioGroup.Root defaultValue="day" onValueChange={(value) => console.log(value)}>
  <SegmentedControlRadioGroup.Item value="day">Day</SegmentedControlRadioGroup.Item>
  <SegmentedControlRadioGroup.Item value="week">Week</SegmentedControlRadioGroup.Item>
  <SegmentedControlRadioGroup.Item value="month">Month</SegmentedControlRadioGroup.Item>
</SegmentedControlRadioGroup.Root>;
```

## API reference

### SegmentedControlRadioGroup.Root

State container for the group; renders a hidden radio input for form integration.

<PropsTable component="SegmentedControlRadioGroup.Root" />

### SegmentedControlRadioGroup.Item

An individual selectable segment.

<PropsTable component="SegmentedControlRadioGroup.Item" />
