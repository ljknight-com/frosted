# IconButton

Trigger an action with a square, icon-only button.

<Demo id="icon-button" />

## Usage

```tsx
import { IconButton } from '@aussieljk/frosted';

<IconButton variant="soft" aria-label="Search">
  <SearchIcon />
</IconButton>;
```

Always provide an `aria-label` — there is no visible text to name the control.

## Props

IconButton shares Button's sizes, variants, and colors, and extends the native `button` element.

<PropsTable component="IconButton" />
