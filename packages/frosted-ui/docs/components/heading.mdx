# Heading

Display a semantic heading with consistent styling.

<Demo id="heading" />

## Usage

```tsx
import { Heading } from '@aussieljk/frosted';

<Heading size="6">Typographic principles</Heading>;
```

## Props

Heading renders an `h1` by default — use the `render` prop to render another heading level (e.g. `render={<h2 />}`).
