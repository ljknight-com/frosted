# TabsNav

Navigate between pages with a link-based tab bar.

<Demo id="tabs-nav" />

## Usage

```tsx
import { TabsNav } from '@aussieljk/frosted';

<TabsNav.Root>
  <TabsNav.Link active href="/account">
    Account
  </TabsNav.Link>
  <TabsNav.Link href="/settings">Settings</TabsNav.Link>
</TabsNav.Root>;
```

Use the `render` prop on `TabsNav.Link` to integrate with your framework's router, e.g.
`render={<NextLink href="/account" />}`.

## API reference

### TabsNav.Root

Container for the tab links.

### TabsNav.Link
