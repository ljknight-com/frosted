# ZStack

Layer children on top of one another, back to front.

<Demo id="z-stack" />

## Usage

```tsx
import { ZStack } from '@aussieljk/frosted';

<ZStack alignment="bottomTrailing">
  <img src="photo.jpg" alt="Photo" />
  <Badge>Live</Badge>
</ZStack>;
```

## Props

ZStack extends the native `div` element — standard div attributes are also supported.
Later children render in front of earlier ones.

<PropsTable component="ZStack" />

## Related

`Overlay` layers content in front of a base view without affecting its layout;
`HStack` and `VStack` for one-dimensional layouts.
