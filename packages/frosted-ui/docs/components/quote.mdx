# Quote

Mark a short inline quotation.

<Demo id="quote" />

## Usage

```tsx
import { Quote, Text } from '@aussieljk/frosted';

<Text>
  His famous quote, <Quote>Good design is a language, not a style</Quote>.
</Text>;
```

## Props

Quote renders the native `q` element and has no props of its own — standard `q` attributes are supported.

## Related

`Blockquote` for block-level quotations.
