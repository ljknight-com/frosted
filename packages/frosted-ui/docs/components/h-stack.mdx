# HStack

Lay out children in a horizontal row, SwiftUI-style.

<Demo id="h-stack" />

## Usage

```tsx
import { HStack } from '@aussieljk/frosted';

<HStack spacing={8} alignment="center">
  <Avatar fallback="JD" />
  <Text>Jane Doe</Text>
</HStack>;
```

## Props

HStack extends the native `div` element — standard div attributes are also supported.

<PropsTable component="HStack" />

## Related

`VStack` for vertical layouts, `ZStack` for layered ones, and `Spacer` to push
siblings apart within a stack.
