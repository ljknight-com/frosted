# Accordion

Expand and collapse sections of content, one or many at a time.

<Demo id="accordion" />

## Usage

```tsx
import { Accordion } from '@aussieljk/frosted';

<Accordion.Root type="single" defaultValue={['item-1']}>
  <Accordion.Item value="item-1">
    <Accordion.Trigger>Is it accessible?</Accordion.Trigger>
    <Accordion.Content>Yes. It adheres to the WAI-ARIA design pattern.</Accordion.Content>
  </Accordion.Item>
</Accordion.Root>;
```

## API reference

### Accordion.Root

State container for the items; `type` controls whether one or multiple panels can be open.

<PropsTable component="Accordion.Root" />

### Accordion.Item

A single collapsible section, identified by `value`.

<PropsTable component="Accordion.Item" />

### Accordion.Trigger

Toggles its item open and closed.

<PropsTable component="Accordion.Trigger" />

### Accordion.Content

The collapsible panel for its item.

<PropsTable component="Accordion.Content" />
