# Slider

Select a value, or a range of values, along a track.

<Demo id="slider" />

## Usage

```tsx
import { Slider } from '@aussieljk/frosted';

<Slider defaultValue={[50]} min={0} max={100} step={5} />;
```

Pass two values for a range slider: `defaultValue={[25, 75]}`.
