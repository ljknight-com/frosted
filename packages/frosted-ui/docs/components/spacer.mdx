# Spacer

Push siblings apart along a stack's axis.

<Demo id="spacer" />

## Usage

```tsx
import { HStack, Spacer } from '@aussieljk/frosted';

<HStack>
  <Logo />
  <Spacer minLength={16} />
  <Button>Sign in</Button>
</HStack>;
```

## Props

Spacer extends the native `div` element — it is a flexible, invisible element that
expands along its stack's main axis, like SwiftUI's `Spacer`.

<PropsTable component="Spacer" />

## Related

`HStack` and `VStack` provide the axis a Spacer expands along.
