# DatePicker

Pick a date from a segmented field with a popover calendar.

<Demo id="date-picker" />

## Usage

Date values come from [`@internationalized/date`](https://react-spectrum.adobe.com/internationalized/date/):

```tsx
import { DatePicker } from '@aussieljk/frosted';
import { parseDate } from '@internationalized/date';

<DatePicker
  aria-label="Appointment date"
  defaultValue={parseDate('2020-02-03')}
  minValue={parseDate('2020-01-01')}
  maxValue={parseDate('2020-12-31')}
/>;
```

## Related

`DateField` is the keyboard-only field without the popover, `DateRangePicker` selects a start and end date, and
`Calendar` is the standalone month grid for building custom pickers.
