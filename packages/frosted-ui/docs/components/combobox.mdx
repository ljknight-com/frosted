# Combobox

Pick one or more options from a filterable list.

<Demo id="combobox" />

## Usage

Combobox is a compound component. Unlike Autocomplete, the value always comes from the list — the input only
filters it:

```tsx
import { Combobox, ScrollArea } from '@aussieljk/frosted';

<Combobox.Root items={fruits} defaultValue="Apple">
  <Combobox.InputRoot showClear>
    <Combobox.Input placeholder="Choose a fruit..." />
  </Combobox.InputRoot>
  <Combobox.Content>
    <ScrollArea type="auto" style={{ maxHeight: 300 }}>
      <Combobox.Empty>No fruits found.</Combobox.Empty>
      <Combobox.List>
        {(item) => (
          <Combobox.Item key={item} value={item}>
            {item}
          </Combobox.Item>
        )}
      </Combobox.List>
    </ScrollArea>
  </Combobox.Content>
</Combobox.Root>;
```

Pass `multiple` to `Combobox.Root` and swap the input for `Combobox.Chips` + `Combobox.ChipsInput` to select
several values as removable chips.
