# Drawer

A swipeable panel that slides up from the bottom of the screen.

<Demo id="drawer" />

## Usage

```tsx
import { Button, Drawer } from '@aussieljk/frosted';

<Drawer.Root>
  <Drawer.Trigger>
    <Button>Edit profile</Button>
  </Drawer.Trigger>
  <Drawer.Content>
    <Drawer.Header>
      <Drawer.Title>Edit profile</Drawer.Title>
    </Drawer.Header>
    <Drawer.Body>
      …
      <Drawer.Close>
        <Button>Save</Button>
      </Drawer.Close>
    </Drawer.Body>
  </Drawer.Content>
</Drawer.Root>;
```

## API reference

### Drawer.Root

State container for the drawer.

<PropsTable component="Drawer.Root" />

### Drawer.Trigger

Opens the drawer; wraps your own element.

<PropsTable component="Drawer.Trigger" />

### Drawer.Content

The sliding panel. Structure its children with `Drawer.Header`, `Drawer.Body`, and
optionally `Drawer.StickyFooter`.

<PropsTable component="Drawer.Content" />

### Drawer.Title

Accessible title, rendered inside `Drawer.Header`.

<PropsTable component="Drawer.Title" />

### Drawer.Close

Closes the drawer; wraps your own element.

<PropsTable component="Drawer.Close" />
