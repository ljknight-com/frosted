# OTP Field

Enter a one-time password or verification code, one character per slot.

<Demo id="otp-field" />

## Usage

```tsx
import { OTPField } from '@aussieljk/frosted';

<OTPField.Root
  maxLength={6}
  render={({ slots }) => (
    <>
      <OTPField.Group>
        {slots.slice(0, 3).map((slot, index) => (
          <OTPField.Slot key={index} {...slot} />
        ))}
      </OTPField.Group>
      <OTPField.Separator />
      <OTPField.Group>
        {slots.slice(3).map((slot, index) => (
          <OTPField.Slot key={index} {...slot} />
        ))}
      </OTPField.Group>
    </>
  )}
/>;
```

## API reference

### OTPField.Root

The hidden input and state container. Its `render` prop receives the slot states.

<PropsTable component="OTPField.Root" />

### OTPField.Group

Visually groups a run of slots.

<PropsTable component="OTPField.Group" />

### OTPField.Slot

Displays a single character of the code. Spread the corresponding slot state from `render` into it.

<PropsTable component="OTPField.Slot" />

### OTPField.Separator

A visual divider between groups.
