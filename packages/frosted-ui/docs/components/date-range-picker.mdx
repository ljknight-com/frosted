# DateRangePicker

Pick a start and end date from a popover calendar.

<Demo id="date-range-picker" />

## Usage

The value is a `{ start, end }` pair of dates from
[`@internationalized/date`](https://react-spectrum.adobe.com/internationalized/date/):

```tsx
import { DateRangePicker } from '@aussieljk/frosted';
import { parseDate } from '@internationalized/date';

<DateRangePicker
  aria-label="Trip dates"
  defaultValue={{
    start: parseDate('2020-02-03'),
    end: parseDate('2020-02-08'),
  }}
/>;
```
