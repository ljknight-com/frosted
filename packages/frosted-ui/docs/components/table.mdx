# Table

Display tabular data in semantic rows and columns.

<Demo id="table" />

## Usage

```tsx
import { Table } from '@aussieljk/frosted';

<Table.Root variant="surface">
  <Table.Table>
    <Table.Header>
      <Table.Row>
        <Table.ColumnHeaderCell>Name</Table.ColumnHeaderCell>
        <Table.ColumnHeaderCell>Email</Table.ColumnHeaderCell>
      </Table.Row>
    </Table.Header>
    <Table.Body>
      <Table.Row>
        <Table.RowHeaderCell>Ada Lovelace</Table.RowHeaderCell>
        <Table.Cell>ada@example.com</Table.Cell>
      </Table.Row>
    </Table.Body>
  </Table.Table>
</Table.Root>;
```

## API reference

### Table.Root

The styled container (a `<div>`, not the `<table>` itself); provides the size and variant styling.

<PropsTable component="Table.Root" />

### Table.Table

The `<table>` element itself; must be placed inside `Table.Root`.

<PropsTable component="Table.Table" />

### Table.Header, Table.Body, Table.Footer

Semantic sections (`<thead>`, `<tbody>`, `<tfoot>`); they accept the corresponding native attributes.

### Table.Row

A table row (`<tr>`); use `align` to control the vertical alignment of its cells.

<PropsTable component="Table.Row" />

### Table.Cell

A standard data cell (`<td>`).

<PropsTable component="Table.Cell" />

### Table.ColumnHeaderCell

A header cell for a column (`<th scope="col">`), used inside `Table.Header` rows.

<PropsTable component="Table.ColumnHeaderCell" />

### Table.RowHeaderCell

A header cell for a row (`<th scope="row">`), typically the row's identifying column.

<PropsTable component="Table.RowHeaderCell" />

### Table.ColumnHeaderCellButton

A ghost button for column headers with built-in sortable styling. Toggle `sortDirection` from your own
sorting state — it is presentational and does not sort data itself.

<PropsTable component="Table.ColumnHeaderCellButton" />

## Related
