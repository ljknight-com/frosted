# VStack

Lay out children in a vertical column, SwiftUI-style.

<Demo id="v-stack" />

## Usage

```tsx
import { VStack } from '@aussieljk/frosted';

<VStack alignment="leading" spacing={12}>
  <Heading>Title</Heading>
  <Text>Description</Text>
</VStack>;
```

## Props

VStack extends the native `div` element — standard div attributes are also supported.

<PropsTable component="VStack" />

## Related

`HStack` for horizontal layouts, `ZStack` for layered ones, and `Spacer` to push
siblings apart within a stack.
