# Number Field

Enter a number, with increment and decrement buttons.

<Demo id="number-field" />

## Usage

```tsx
import { NumberField } from '@aussieljk/frosted';

<NumberField.Root defaultValue={50} min={0} max={100}>
  <NumberField.Input />
</NumberField.Root>;
```

## API reference

### NumberField.Root

State container for the field — holds the value, bounds, formatting, and the `buttonLayout`.

<PropsTable component="NumberField.Root" />

### NumberField.Input

The numeric input.

### NumberField.Slot

Decorations (icons, units) inside the input area.

<PropsTable component="NumberField.Slot" />

### NumberField.Increment

Custom increment button — the default layouts render one automatically.

<PropsTable component="NumberField.Increment" />

### NumberField.Decrement

Custom decrement button — the default layouts render one automatically.
