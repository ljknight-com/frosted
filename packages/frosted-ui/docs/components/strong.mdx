# Strong

Mark text with strong importance.

<Demo id="strong" />

## Usage

```tsx
import { Strong, Text } from '@aussieljk/frosted';

<Text>
  The most important thing to remember is, <Strong>stay positive</Strong>.
</Text>;
```

## Props

Strong renders the native `strong` element and has no props of its own — standard `strong` attributes are supported.

## Related

`Em` for stress emphasis rather than strong importance.
