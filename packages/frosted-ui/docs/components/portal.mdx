# Portal

Render children into a different part of the DOM.

<Demo id="portal" />

## Usage

```tsx
import { Portal } from '@aussieljk/frosted';

<Portal>
  <Toast>Saved!</Toast>
</Portal>;
```

## Props

Portal is SSR-safe: it renders nothing until mounted on the client, then appends to
`document.body` unless a `container` is provided. Children are wrapped in a `div` by
default (override via the `render` prop).
