# Skeleton

Show a pulsing placeholder while content is loading.

<Demo id="skeleton" />

## Usage

```tsx
import { Skeleton } from '@aussieljk/frosted';

<Skeleton.Avatar size="3" />
<Skeleton.Text size="2" style={{ width: 200 }} />
<Skeleton.Rect style={{ height: 96 }} />;
```

All skeletons on the page pulse in phase, regardless of when they mount.

## API reference

### Skeleton.Avatar

A placeholder shaped like an Avatar of the same `size` and `shape`.

<PropsTable component="Skeleton.Avatar" />

### Skeleton.Text

A placeholder for a line of Text of the same `size`.

<PropsTable component="Skeleton.Text" />

### Skeleton.Rect

A rectangular placeholder; size it via `className` or `style`.

<PropsTable component="Skeleton.Rect" />
