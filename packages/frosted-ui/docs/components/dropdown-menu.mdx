# Dropdown Menu

A menu of actions opened from a trigger button.

<Demo id="dropdown-menu" />

## Usage

```tsx
import { Button, DropdownMenu } from '@aussieljk/frosted';

<DropdownMenu.Root>
  <DropdownMenu.Trigger>
    <Button variant="soft">Options</Button>
  </DropdownMenu.Trigger>
  <DropdownMenu.Content>
    <DropdownMenu.Item shortcut="⌘ E">Edit</DropdownMenu.Item>
    <DropdownMenu.Item shortcut="⌘ D">Duplicate</DropdownMenu.Item>
    <DropdownMenu.Separator />
    <DropdownMenu.Item color="danger">Delete</DropdownMenu.Item>
  </DropdownMenu.Content>
</DropdownMenu.Root>;
```

## API reference

### DropdownMenu.Root

State container for the menu.

<PropsTable component="DropdownMenu.Root" />

### DropdownMenu.Trigger

Opens the menu; wraps your own element.

<PropsTable component="DropdownMenu.Trigger" />

### DropdownMenu.Content

The menu panel containing the items.

<PropsTable component="DropdownMenu.Content" />

### DropdownMenu.Item

A single action in the menu.

<PropsTable component="DropdownMenu.Item" />

### DropdownMenu.CheckboxItem

A toggleable item with a checkmark.

<PropsTable component="DropdownMenu.CheckboxItem" />

### DropdownMenu.RadioItem

A single-choice item; use inside `DropdownMenu.RadioGroup`.

<PropsTable component="DropdownMenu.RadioItem" />

### DropdownMenu.Sub

State container for a nested submenu; contains `DropdownMenu.SubTrigger` and
`DropdownMenu.SubContent`.

<PropsTable component="DropdownMenu.Sub" />
