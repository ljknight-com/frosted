# Form

Submit a group of fields with consolidated error handling.

<Demo id="form" />

## Usage

```tsx
import { Button, Field, Form, TextField } from '@aussieljk/frosted';

<Form
  onSubmit={(event) => {
    event.preventDefault();
    // handle submission
  }}
>
  <Field.Root name="email">
    <Field.Label>Email</Field.Label>
    <TextField.Root>
      <TextField.Input type="email" required />
    </TextField.Root>
    <Field.Error match="valueMissing">Email is required</Field.Error>
  </Field.Root>
  <Button type="submit">Subscribe</Button>
</Form>;
```

Pass server-side validation errors via the `errors` prop — keys match each `Field.Root` `name`.

## Props

Form extends the native `form` element — standard form attributes are also supported.
