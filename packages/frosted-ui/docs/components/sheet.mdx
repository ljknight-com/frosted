# Sheet

A panel that slides in from the edge of the screen.

<Demo id="sheet" />

## Usage

```tsx
import { Button, Sheet } from '@aussieljk/frosted';

<Sheet.Root>
  <Sheet.Trigger>
    <Button>Edit profile</Button>
  </Sheet.Trigger>
  <Sheet.Content>
    <Sheet.Header>
      <Sheet.Title>Edit profile</Sheet.Title>
      <Sheet.Description>Make changes to your profile.</Sheet.Description>
    </Sheet.Header>
    <Sheet.Body>
      …
      <Sheet.Close>
        <Button>Save</Button>
      </Sheet.Close>
    </Sheet.Body>
  </Sheet.Content>
</Sheet.Root>;
```
