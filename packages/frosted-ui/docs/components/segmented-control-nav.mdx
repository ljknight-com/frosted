# SegmentedControlNav

Navigate between pages with a filled, link-based segmented group.

<Demo id="segmented-control-nav" />

## Usage

```tsx
import { SegmentedControlNav } from '@aussieljk/frosted';

<SegmentedControlNav.Root>
  <SegmentedControlNav.Link active href="/account">
    Account
  </SegmentedControlNav.Link>
  <SegmentedControlNav.Link href="/settings">Settings</SegmentedControlNav.Link>
</SegmentedControlNav.Root>;
```

Use the `render` prop on `SegmentedControlNav.Link` to integrate with your framework's
router, e.g. `render={<NextLink href="/account" />}`.

## API reference

### SegmentedControlNav.Root

Container for the segment links.

<PropsTable component="SegmentedControlNav.Root" />

### SegmentedControlNav.Link

An individual navigation link; mark the current page with `active`.

<PropsTable component="SegmentedControlNav.Link" />
