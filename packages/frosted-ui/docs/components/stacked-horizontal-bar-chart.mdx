# Stacked Horizontal Bar Chart

Divide a single horizontal bar into colored segments proportional to their values.

<Demo id="stacked-horizontal-bar-chart" />

## Usage

```tsx
import { StackedHorizontalBarChart } from '@aussieljk/frosted';

<StackedHorizontalBarChart
  data={[
    { label: 'Completed', value: 60, color: 'green' },
    { label: 'In progress', value: 25, color: 'blue' },
    { label: 'Failed', value: 15, color: 'red' },
  ]}
/>;
```

Each segment's width is its `value` as a percentage of the sum of all values. Hovering a segment shows
a tooltip with its label, and each segment carries an `aria-label` combining the label and percentage.
