# Progress

Visualize a value against a maximum as a horizontal bar.

<Demo id="progress" />

## Usage

```tsx
import { Progress } from '@aussieljk/frosted';

<Progress value={40} max={100} aria-label="40% complete" />;
```

`value` defaults to 0 and `max` to 100; the indicator width is clamped to 0–100% of `max`.

## Props

Progress is built on Base UI's `Meter` and also accepts its root props (`value`, `max`, …).

<PropsTable component="Progress" />

## Related

`CircularProgress` for a ring-shaped indicator and `Spinner` for indeterminate loading.
