# DateField

Type a date with segmented keyboard input.

<Demo id="date-field" />

## Usage

Date values come from [`@internationalized/date`](https://react-spectrum.adobe.com/internationalized/date/) — use
`parseDate` (or `parseZonedDateTime` for date-times) to construct them:

```tsx
import { DateField } from '@aussieljk/frosted';
import { parseDate } from '@internationalized/date';

<DateField aria-label="Birth date" defaultValue={parseDate('2020-02-03')} />;
```

## Related
