# Context Menu

A menu of actions opened by right-clicking (or long-pressing) an area.

<Demo id="context-menu" />

## Usage

```tsx
import { ContextMenu } from '@aussieljk/frosted';

<ContextMenu.Root>
  <ContextMenu.Trigger>
    <div>Right-click here</div>
  </ContextMenu.Trigger>
  <ContextMenu.Content>
    <ContextMenu.Item shortcut="⌘ E">Edit</ContextMenu.Item>
    <ContextMenu.Item shortcut="⌘ D">Duplicate</ContextMenu.Item>
    <ContextMenu.Separator />
    <ContextMenu.Item color="danger">Delete</ContextMenu.Item>
  </ContextMenu.Content>
</ContextMenu.Root>;
```

## API reference

### ContextMenu.Root

State container for the menu; wrap it around the right-clickable area (the
`ContextMenu.Trigger`).

<PropsTable component="ContextMenu.Root" />

### ContextMenu.Content

The menu panel containing the items.

<PropsTable component="ContextMenu.Content" />

### ContextMenu.Item

A single action in the menu.

<PropsTable component="ContextMenu.Item" />

### ContextMenu.CheckboxItem

A toggleable item with a checkmark.

<PropsTable component="ContextMenu.CheckboxItem" />

### ContextMenu.RadioItem

A single-choice item; use inside `ContextMenu.RadioGroup`.

<PropsTable component="ContextMenu.RadioItem" />

### ContextMenu.Sub

State container for a nested submenu; contains `ContextMenu.SubTrigger` and
`ContextMenu.SubContent`.

<PropsTable component="ContextMenu.Sub" />
