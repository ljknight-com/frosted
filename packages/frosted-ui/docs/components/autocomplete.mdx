# Autocomplete

Suggest matching options while typing free-form text.

<Demo id="autocomplete" />

## Usage

Autocomplete is a compound component. The input stays free-form — suggestions only optionally complete the text
(use Combobox when the value must come from the list). `Autocomplete.Input` renders into a `TextField.Input` so the
field looks like every other text field:

```tsx
import { Autocomplete, ScrollArea, TextField } from '@aussieljk/frosted';

<Autocomplete.Root items={tags}>
  <TextField.Root>
    <Autocomplete.Input render={<TextField.Input placeholder="Search tags..." />} />
  </TextField.Root>
  <Autocomplete.Content>
    <ScrollArea type="auto">
      <Autocomplete.Empty>No tags found.</Autocomplete.Empty>
      <Autocomplete.List>
        {(tag) => (
          <Autocomplete.Item key={tag as string} value={tag}>
            {tag as string}
          </Autocomplete.Item>
        )}
      </Autocomplete.List>
    </ScrollArea>
  </Autocomplete.Content>
</Autocomplete.Root>;
```

## API reference

### Autocomplete.Root

State container: holds the `items`, the input `value`, and open state.

<PropsTable component="Autocomplete.Root" />

### Autocomplete.Input

The text input; pass `render` to compose it with `TextField.Input`.

<PropsTable component="Autocomplete.Input" />

### Autocomplete.Content

The popup that anchors to the input (or to a `TextField.Root` via `anchor`).

<PropsTable component="Autocomplete.Content" />

### Autocomplete.List

Renders each filtered item through its function child.

<PropsTable component="Autocomplete.List" />

### Autocomplete.Item

A single suggestion.

<PropsTable component="Autocomplete.Item" />

### Other parts

`Autocomplete.Empty` shows when nothing matches, `Autocomplete.Group` / `Autocomplete.GroupLabel` add section
headings, `Autocomplete.Clear` clears the input, `Autocomplete.Trigger` opens the popup from an external button
(e.g. an emoji picker), `Autocomplete.Row` lays out grid rows, and `Autocomplete.Separator` divides sections.

## Related

`Combobox` when the selection must come from the list, and `Select` for a plain non-filterable dropdown.
