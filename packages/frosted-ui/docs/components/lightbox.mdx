# Lightbox

Open images in a fullscreen, navigable overlay.

<Demo id="lightbox" />

## Usage

```tsx
import { Lightbox } from '@aussieljk/frosted';

<Lightbox.Root viewTransition>
  <Lightbox.Trigger index={0} crossfade>
    <img src={thumb} alt="…" />
  </Lightbox.Trigger>

  <Lightbox.Content aria-label="Photo gallery">
    <Lightbox.Close>×</Lightbox.Close>
    <Lightbox.ItemGroup>
      <Lightbox.Item index={0}>
        <img src={full} alt="…" />
      </Lightbox.Item>
    </Lightbox.ItemGroup>
    <Lightbox.Previous>‹</Lightbox.Previous>
    <Lightbox.Next>›</Lightbox.Next>
  </Lightbox.Content>
</Lightbox.Root>;
```

## API reference

Beyond the core parts below, the stories show `Lightbox.Caption` for per-item captions, `Lightbox.Thumbnail` / `Lightbox.ThumbnailGroup` for a filmstrip, and `Lightbox.Zoom` / `Lightbox.ZoomIn` / `Lightbox.ZoomOut` for pinch-and-button zooming.

### Lightbox.Root

State container; pairs each `Trigger` with the `Item` of the same `index`.

<PropsTable component="Lightbox.Root" />

### Lightbox.Trigger

Opens the lightbox at its `index`; `crossfade` morphs the trigger into the item.

<PropsTable component="Lightbox.Trigger" />

### Lightbox.Content

The fullscreen overlay.

### Lightbox.ItemGroup

Container for the items; `preload` eagerly mounts neighbors.

### Lightbox.Item

One slide, matched to the trigger with the same `index`.
