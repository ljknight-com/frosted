# Text

Display a run of foundational text content.

<Demo id="text" />

## Usage

```tsx
import { Text } from '@aussieljk/frosted';

<Text size="3" weight="medium">
  The quick brown fox jumps over the lazy dog.
</Text>;
```

## Props

Text renders a `span` by default — use the `render` prop to render another element (e.g. a `label` or `p`).

<PropsTable component="Text" />

## Related

`Heading` for section titles, and `Em`, `Strong`, `Code`, `Kbd`, `Quote`, and `Link` for inline formatting within text.
