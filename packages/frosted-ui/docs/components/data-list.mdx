# Data List

Present label/value metadata as a definition list.

<Demo id="data-list" />

## Usage

```tsx
import { Badge, DataList } from '@aussieljk/frosted';

<DataList.Root>
  <DataList.Item>
    <DataList.Label>Status</DataList.Label>
    <DataList.Value>
      <Badge color="success">Active</Badge>
    </DataList.Value>
  </DataList.Item>
</DataList.Root>;
```

## API reference

### DataList.Root

The definition list (`<dl>`) container; sets the size and orientation of every row.

<PropsTable component="DataList.Root" />

### DataList.Item

One row of the list, containing a `Label` and a `Value`.

<PropsTable component="DataList.Item" />

### DataList.Label

The term (`<dt>`) of a row, rendered as a muted label.

<PropsTable component="DataList.Label" />

### DataList.Value

The description (`<dd>`) of a row, holding the value content. Accepts the native `dd` attributes.
