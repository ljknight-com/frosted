# Separator

Divide sections of content or groups of items.

<Demo id="separator" />

## Usage

```tsx
import { Separator } from '@aussieljk/frosted';

<Separator size="4" />;

// Vertical line
<Separator orientation="vertical" />;
```

## Props

Separator wraps Base UI's Separator primitive (rendered with `role="separator"`).
