# Link

Navigate to another page or resource.

<Demo id="link" />

## Usage

```tsx
import { Link } from '@aussieljk/frosted';

<Link href="/docs" underline="hover">
  Documentation
</Link>;
```

## Props

Link renders the native `a` element — standard anchor attributes are also supported, and the `render` prop lets you compose it with a router's link component.
