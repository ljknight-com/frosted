@aussieljk/frosted includes SwiftUI-style layout primitives. They take layout-specific props
(`spacing`, `alignment`, …) and, like every @aussieljk/frosted component, also accept `className` and
`style` for anything else.

<Demo name="layout" />

## HStack

Lays children out horizontally. `spacing` is the gap in pixels; `alignment` controls the
cross-axis: `top`, `center` (default), `bottom`, `firstTextBaseline`, `lastTextBaseline`.

```tsx
import { Badge, Button, HStack } from '@aussieljk/frosted';

<HStack spacing={12} alignment="center">
  <Button>Save</Button>
  <Badge color="green">Synced</Badge>
</HStack>;
```

## VStack

Lays children out vertically. `alignment` controls the cross-axis: `leading`, `center`
(default), `trailing`.

```tsx
import { Heading, Text, VStack } from '@aussieljk/frosted';

<VStack spacing={8} alignment="leading">
  <Heading size="4">Billing</Heading>
  <Text size="2" color="gray">
    Manage your subscription and invoices.
  </Text>
</VStack>;
```

## ZStack

Stacks children on top of each other. `alignment` positions them using SwiftUI-style values:
`center` (default), `leading`, `trailing`, `top`, `bottom`, `topLeading`, `topTrailing`,
`bottomLeading`, `bottomTrailing`, plus text-baseline variants.

```tsx
import { Avatar, Badge, ZStack } from '@aussieljk/frosted';

<ZStack alignment="bottomTrailing">
  <Avatar size="5" fallback="LK" />
  <Badge color="green" size="1">
    on
  </Badge>
</ZStack>;
```

## Grid and Grid.Row

A SwiftUI-style grid: the column count is the largest number of cells in any row, and cells
align across rows.

```tsx
import { Grid, Text } from '@aussieljk/frosted';

<Grid horizontalSpacing={16} verticalSpacing={8}>
  <Grid.Row>
    <Text>1 × 1</Text>
    <Text>1 × 2</Text>
  </Grid.Row>
  <Grid.Row>
    <Text>2 × 1</Text>
    <Text>2 × 2</Text>
  </Grid.Row>
</Grid>;
```

<PropsTable component="Grid" />

`Grid.Row` takes no extra props beyond the standard `div` attributes.

## Spacer

Expands to fill the available space along the parent stack's axis — the SwiftUI way to push
siblings apart. `minLength` sets the minimum size in pixels.

```tsx
import { Button, Heading, HStack, Spacer } from '@aussieljk/frosted';

<HStack spacing={8}>
  <Heading size="4">Members</Heading>
  <Spacer />
  <Button size="1">Invite</Button>
</HStack>;
```

## Overlay

Layers `Overlay.Content` children in front of the remaining children, matching SwiftUI's
`.overlay(alignment:)` semantics:

```tsx
import { Card, Overlay, Spinner } from '@aussieljk/frosted';

<Overlay alignment="center">
  <Card size="3">Loading…</Card>
  <Overlay.Content>
    <Spinner />
  </Overlay.Content>
</Overlay>;
```

<PropsTable component="Overlay" />

`Overlay.Content` takes no extra props beyond the standard `div` attributes.

## Escape hatches

Every layout component forwards standard `div` props, so `className` (e.g. Tailwind utilities)
and `style` compose naturally with the SwiftUI-style props:

```tsx
<HStack spacing={12} className="rounded-lg border p-4">
  …
</HStack>
```
